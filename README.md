# Siem-Elastic-Stack-Project
A full SIEM lab using Elastic Stack 8.11; deployed Fleet-managed agents, ingested multi-source logs (ECS-normalised), simulated SSH brute-force (MITRE ATT&amp;CK T1110.001) with Hydra, and engineered KQL threshold detection rules achieving live adversary attribution. Designed role-differentiated dashboards for SOC analysts and executive stakeholders.
